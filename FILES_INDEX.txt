КЛЮЧЕВЫЕ КОНСТАНТЫ:
===================

Правила повторения (Recurrence.rule):
- daily (каждый день)
- weekly (каждую неделю)
- monthly (каждый месяц)
- every_n_days (каждые N дней)

Типы задач (Task.task_type):
- one_time (разовая задача)
- recurring (повторяющаяся задача)
- deadline (дедлайн с распределением времени)

Формат даты-времени:
- DATE_TIME_FORMAT = "%d.%m.%Y %H:%M"
- Пример: "24.05.2026 18:30"

Job ID формат в APScheduler:
- f"task:{task_id}:offset:{offset}"
- Пример: "task:1:offset:15"

Формат напоминаний:
- Целые числа, разделённые запятыми
- Пример: "15,30,60" (за 15, 30, 60 минут)

========================

Проблема #1: Напоминания не переопланируются для повторяющихся задач

Проблема #2: APScheduler не персистирует jobs
Решение: Использовать SQLAlchemy job store или кастомное решение

Проблема #3: Нет управления уведомлениями
Решение: Добавить UserNotificationSettings модель

Проблема #4: Нет учёта летнего/зимнего времени
Решение: Использовать ZoneInfo для автоматического преобразования

Проблема #5: Job ID может конфликтовать при изменении напоминаний
Решение: Удалять старые jobs перед созданием новых
